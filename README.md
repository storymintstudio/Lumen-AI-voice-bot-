# Lumen AI Bot Landing Page

A modern and responsive landing page for **Lumen AI Bot**, built with **HTML, CSS, and JavaScript**. It showcases an AI-powered voice-to-text solution with a clean UI, smooth interactions, and a mobile-friendly design.

## Features

- Responsive design
- Modern and clean interface
- Voice-to-text focused landing page
- Smooth animations
- Built with HTML, CSS, and JavaScript

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/lumen-ai-landing-page.git
   ```
2. Open `index.html` in your browser.

## Tech Stack

- HTML5
- CSS3
- JavaScript
